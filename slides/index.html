GraphQL

  • alternative à REST
  • language de requêtage
  • fortement typé
  • définir précisément les éléments voulus
  • point d'accès unique (ex: /graphql)

Requête simple


query {
    userCount
}

Payload JSON

{"query":"query {\n\tuserCount\n}"}

Réponse

{
  "data": {
    "userCount": 10
  }
}

Modèle

Requête complexe

multi-requête, paramètres et champs


query {
	userCount
	usersPaginated(page:1,count:5,sortedBy:"name"){
		name
		company{
			name
		}
	}
}
