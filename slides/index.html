GraphQL

  • alternative à REST
  • language de requêtage
  • fortement typé
  • filtrer la réponse
  • point d'accès unique (ex: /graphql) GET ou POST

Requête simple


query {
    userCount
}

Payload JSON

{"query":"query {\n\tuserCount\n}"}

Réponse

{
  "data": {
    "userCount": 10
  }
}

Requête complexe*

multi-requête, paramètres et champs


query UsersPaginated($page:Int, $count:Int, $sortedBy:String) {
	userCount
	usersPaginated(page:$page,count:$count,sortedBy:$sortedBy){
		id
		name
		email
		company{
			name
		}
	}
}

* GraphQL ne fait pas la pagination
